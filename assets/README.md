# Assets Directory

## Rain Logo

To add the Rain logo to your PDF reports:

1. Save the Rain logo as `rain-logo.png` in this directory
2. Recommended dimensions: 200x80 pixels or similar aspect ratio
3. The logo will appear on the top left of the PDF title page

### File Location
The logo should be saved at:
```
\\wsl.localhost\Ubuntu\home\laurencesher\projects\changereport\assets\rain-logo.png
```

### Logo Specifications
- Format: PNG (transparent background recommended)
- Color: Blue (#2679BD) and Gray (#BBBBBB)
- The logo will be displayed at 100px width in the PDF
